# Mostlylucid.StyloSpam

Planned home for the StyloSpam email detection and proxy runtime.

Initial planned subareas:

- `Gateway/` - SMTP/API proxy runtime (YARP-style "Email Proxy mode")
- `Ingestion/` - message ingestion adapters
- `Orchestration/` - blackboard detection pipeline
- `ContributingDetectors/` - detector contributors
- `Campaigns/` - similarity and campaign graph services
- `Policies/` - action and policy routing
- `Storage/` - event and campaign persistence
- `Api/` - management and query API
- `Dashboard/` - telemetry and analyst UI integration
