13+ detectors analyse every request. You get the full picture.

Heuristics, behavioural analysis, fingerprinting, fast-path reputation caching, and optional LLM escalation. StyloBot treats a visitor as a multi-vector zero-PII signature, then matches that signature across time for higher-resolution decisions. You get probability, confidence, risk bands, detector breakdowns, and reasoned outputs. Block, log, challenge, throttle, or run shadow mode. All open source.

Drop-in integration
YARP proxy or ASP.NET Middleware adds bot intelligence as HTTP headers. Or use the JSON API.
Full transparency
Every request returns scores, confidence, signals, detector breakdowns, and reasoning.
Zero PII
Nothing stored. No IP addresses, no fingerprints, no cookies. GDPR compliant by design.

Capabilities

Forensic analysis
Behavioral pattern recognition for sophisticated bot attacks.
Real-time adaptation
ML models that adapt to new threats as they appear.
Live dashboard
Monitor threats with real-time analytics and actionable insights.
Multi-vector signatures
Same visitor, different IP? We correlate UA, TLS, behavioral, and client-side signals to track identity across network changes.
Time-travel detection
When a visitor crosses a threshold, past requests are re-evaluated with richer cross-request signals, catching what single-request analysis missed.
Continuous learning
Gets smarter with every request. No manual retraining.
